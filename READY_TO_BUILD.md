### Option 2: UTM VM (Local Build) üñ•Ô∏è

**Time:** 30-60 minutes setup + 60-90 minutes build

#### Quick Setup:

```bash
# 1. Install UTM (if not already)
brew install --cask utm

# 2. Download Ubuntu ARM64 ISO (if not already)
cd ~/Downloads
wget https://cdimage.ubuntu.com/releases/22.04/release/ubuntu-22.04.5-live-server-arm64.iso

# 3. Create VM in UTM
open -a UTM
# - Create New VM
# - Select "Virtualize" (NOT Emulate!)
# - Choose Linux
# - Select Ubuntu ISO
# - 8GB RAM, 6 CPU cores, 100GB disk
# - Install Ubuntu Server

# 4. After Ubuntu installation, copy setup script to VM
scp utm-setup-script.sh builder@<VM_IP>:~/

# 5. In Ubuntu VM, run setup
chmod +x utm-setup-script.sh
./utm-setup-script.sh
```

**What will happen:**
1. Updates Ubuntu system
2. Installs build dependencies
3. Downloads Tizen Studio 6.1 (267 MB)
4. Installs toolchain packages
5. Clones Kodi repository
6. Applies C++17 patches
7. Ready to build!

**Then build Kodi:**
```bash
cd ~/kodi-tizen
~/build-kodi.sh  # Auto-generated by setup script
```
